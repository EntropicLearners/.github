# EntropicLearners
 
We build and share open-source LLM-based tools for education and research.
Our work sits at the intersection of large language models, domain-specific knowledge, and practical pedagogy —
exploring cutting-edge AI into tools that students and researchers can actually use.
 
## Projects
 
### [stan](https://github.com/EntropicLearners/stan) — Chemical Engineering Thermodynamics Tutor
 
Stan is an LLM-based learning assistant for single- and multicomponent equilibrium thermodynamics.
It combines large language models with classical chemical engineering software to support both students and instructors.
 
Key features include:
- **Hybrid search** (LLM + regex) over textbooks and lecture recordings
- **Equation-of-state library** for thermodynamic calculations
- **Lecture pipeline** — from raw video recordings to searchable, LLM-queryable transcripts (via Whisper + vector indexing)
- **RAG pipelines** for student Q&A and TOC-aware textbook retrieval
 
Stan is described in our paper: [arxiv:2603.04657](https://arxiv.org/abs/2603.04657)
 
## Philosophy
 
We believe domain-specific LLM tools — built carefully, with subject-matter expertise baked in —
can meaningfully improve how people learn and do research.
We share our work openly so others can build on it.
